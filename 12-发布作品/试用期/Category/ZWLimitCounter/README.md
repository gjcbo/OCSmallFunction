# ZWLimitCounter
限制并统计UITextView输入字数
### 框架使用方法
- 直接把ZWLimitCounter文件夹拖入到您的工程中.
  
  ```objc
  #import "UITextView+ZWLimitCounter.h"
  ``` 
  
- Cocoapods安装
  
  ```objc
  pod search ZWLimitCounter
  pod 'ZWLimitCounter'
  ```
  
  如果发现pod search ZWLimitCounter,然后执行pod repo update --verbose命令更新本地spec缓存（可能需要几分钟），然后再搜索就可以了.
